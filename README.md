# Magpie Repository for Kodi

The official repository for Code-E-Magpie add-ons.

Add the Kodi source:<br/>https://Code-E-Magpie.github.io/repository.magpie/

Use the 'Install from zip file' method to install the Magpie Repository from the Kodi source.

Use the 'Install from repository' method to install add-ons from the Magpie Repository.

For further details of the Magpie Repository, Code-E-Magpie add-ons, Kodi special favourites and Kodi special sources:<br/>https://github.com/Code-E-Magpie/repository.magpie
